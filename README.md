# Pteranodon Tracker

Track your Firmament contribution progress toward the **Castle in the Sky** achievement and the **Pteranodon mount** in Final Fantasy XIV.

## Features

- **Progress Tracker** — Enter your firmament points per crafter and gatherer job. Progress saves automatically to your browser.
- **Gatherables** — Track items you're farming in the Diadem with a searchable item list, per-item targets, and a bulk-add counter.

## Usage

Just open `index.html` in a browser — no build step, no server needed.

Progress is saved to `localStorage`, so it persists between visits in the same browser.

## Jobs tracked

**Crafters:** Carpenter, Blacksmith, Armorer, Goldsmith, Leatherworker, Weaver, Alchemist, Culinarian

**Gatherers:** Miner, Botanist, Fisher

Each job requires 500,000 firmament points · 11 jobs total · 5,500,000 max
